The project combines:

🧠 AI-powered puzzle interpretation using Google Gemini API
🔍 SAT solving using PySAT (Glucose3)
🖼️ Image support with Pillow
🖥️ Simple desktop GUI with Tkinter
Features
Load Sudoku puzzles manually or from images
AI-assisted Sudoku recognition using Gemini API
Solve Sudoku using SAT solver logic
User-friendly Tkinter GUI
Multi-threaded execution to keep UI responsive
JSON export/import support
Technologies Used
Python 3
Tkinter
Google Generative AI (google.generativeai)
PySAT
Pillow (PIL)
JSON
Threading
